HR Data 원큐 Agent 핸즈온 가이드

Copilot Studio Only 버전: Power Automate 없이 Copilot Studio만으로 구현하는 간소화 가이드(8단계)는 여기에서 확인하세요.

개요

목표

사번을 입력하면 HR/평가 데이터를 조회하고, AI를 통해 해당 팀 팀장 포지션의 요구역량과 채점기준에 따라 역량을 분석한 뒤, Word 문서로 사용자 프로필을 생성하는 에이전트를 만듭니다.

완성 구성도

Mock API

  • Base URL: https://mockhrserver.onrender.com
  • 인사정보 조회: GET /api/v1/inhr/employees/{id}
  • 평가/리더십 조회: GET /api/v1/myhr/evaluations/{id}
  • Swagger 문서: https://mockhrserver.onrender.com/docs
  • 직원 100명, 23개 부서

## 분석 절차 (반드시 순서대로 수행)

1단계: 인사정보에서 소속팀과 직급을 확인하세요.

2단계: 참조자료 "Knowledge_포지션별_요구역량.xlsx"에서 해당 소속팀의 행을 찾아 아래 3가지를 확인하세요.
- 핵심직무
- 요구전문성
